#### Work(Quick Action) VS Project 
```

단순 UI나 View, widget 배치같은 경우 work quick action으로 구분
사용자들의 불편사항 중 project를 장기간 프로젝트로 구성

회사가 원하는 USL 보다 높은 이상치를 목표 의지 -> 승인
너무 과해도 적정수준을 찾는다.

목표, 범위, 일정, TFT 구성
```
